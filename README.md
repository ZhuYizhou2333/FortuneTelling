# FortuneTelling
## 简介
这是一个基于紫薇斗数的算命排盘软件。后续会增加更多东方或西方神秘学内容。

项目为清华大学郑莉老师2023秋季开设的课程《c++语言程序设计》大作业。

项目使用c++编写，文件xDate.h与xDate.cpp为网络上的开源代码。

## 当前版本功能概览

当前版本可以实现分类的增、删、导入、导出与命例的新建，删除。

通过选择命例即可实现紫微斗数排盘。

- 改为使用xmake构建
- 重构部分排盘代码
- 使用sqlite数据库存储命例数据
- 可输出文字化的排盘结果
